# SuKeApp
宿客App，基于租房需求的社交App

### 工程基于的框架
#### 1. 网络
retrofit2 + rxAndroid

#### 2. View
rxView

*   注意事项及建议：
    1. networklibrary工程下的assets->netconfig.properties文件最好copy一份到主工程，修改主工程的字段。
    2. 若有语法看不懂，例如
    'bt.setOnClickListener(v->{
        //some code 
     });'
    请看java8 的lambda语法。
